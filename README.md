# 🎬 Movie Recommendation System

This project is a machine learning-based movie recommendation system that suggests similar movies based on user input.

---

## 🚀 Features
- Recommends movies based on similarity
- Uses TF-IDF vectorization
- Calculates similarity using cosine similarity

---

## 🛠️ Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn

---

## 📊 How It Works
- Combined important features like genres, keywords, cast, and director
- Converted text data into vectors using TF-IDF
- Calculated similarity scores using cosine similarity
- Recommended top similar movies

---

## 📁 Dataset
The dataset used in this project is available on Kaggle:

https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata

Download the dataset and place `movies.csv` in the same folder before running the notebook.

---

## ▶️ How to Run
1. Download dataset from Kaggle  
2. Place `movies.csv` in project folder  
3. Open notebook  
4. Run all cells
